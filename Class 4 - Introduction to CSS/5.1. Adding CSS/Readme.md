## Introduction to CSS

#### CSS can be inserted by 3 different ways:
![Alt text](image.png)

### 1ยบ Inline
```
<body>
  <h1 style="color: blue">Style Me in Blue!</h1>
</body>
```
### 2ยบ Internal

```
<head>
  <meta charset="UTF-8">
  <title>Internal</title>
  <style>
    h1 {
      color: red;
    }
  </style>
</head>
```
### 3ยบ External
```
<head>
  <meta charset="UTF-8">
  <title>External</title>
  <link rel="stylesheet" href="./style.css">
</head>
```
